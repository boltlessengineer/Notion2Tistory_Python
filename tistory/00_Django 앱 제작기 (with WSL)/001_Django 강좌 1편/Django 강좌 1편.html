본 강좌는 WSL환경, 즉 Ubuntu 환경을 기준으로 만들어진 강좌입니다

 

이 글은 위 강좌의 WSL 환경용 요약집닙니다. 

 

WSL 에서 Django 웹 개발 환경을 설정해줘야 한다. 

1. 파이썬

Ubuntu에는 python이 기본적으로 설치되어있다. 이 글을 쓰는 현재, 2020-05-04 기준으로 Ubuntu 20.04 LTS에는 python 3.8.2 가 설치돼있다. 

리눅스 OS 버전은 

cat /etc/issue

로, 

파이썬 버전은 

python3 -V

로 확인할 수 있다. ※ V 는 대문자를 써줘야 한다. v 가 아니다. ※ 

 

하지만 파이썬 패키지 설치를 위한 Python Package Index tool는 설치되어있지 않다고 하니 

sudo apt install python3-pip

로 파이썬 패키지 설치툴을 설치해줘야 한다. 

 

2. 가상환경 툴 설치

파이썬 설치가 끝났으면 파이썬 가상환경을 만들어줘야 한다. 위키독스에서 대충 읽었던 기억으로는 아마 여러가지 파이썬 라이브러리 버전과 프로젝트를 관리하기 위해 만든다고 들었던 것 같다. 확실히 파이썬 버전 꼬이면 ※같긴 하지... 

 

위에서 설치해준 pip3로 가상환경을 만들기 위한 virtualenvwrapper를 설치해준다. 

sudo pip3 install virtualenvwrapper

 

그 다음 ~/.zshrc 파일을 열어서 (nano 편집기를 추천한다 nano ~/.zshrc 로 열 수 있다. 

export WORKON_HOME=$HOME/.virtualenvs
export VIRTUALENVWRAPPER_PYTHON=/usr/bin/python3
export VIRTUALENVWRAPPER_VIRTUALENV_ARGS=' -p /usr/bin/python3 '
export PROJECT_HOME=$HOME/Devel
source /usr/local/bin/virtualenvwrapper.sh

아래의 코드를 복붙한다. 

 

터미널을 다시시작 해주면 

virtualenvwrapper.user_scripts creating /home/유저이름/.virtualenvs/쏼라쏼라 같은 글이 여러줄 출력된다. 

이는 에러가 아니므로 안심해도 된다. 

 

3. 가상환경 생성

Django를 설치하기 앞서 가상환경부터 만들어준다. 필수는 아니지만 이렇게 하면 여러개의 프로젝트를 만들 때 라이브러리가 꼬일 일이 없다는 것 같다. 

앞서 설치한 virtualenvwrapper로 가상환경을 만들어 줄 수 있다. 

 

mkvirtualenv my_django_env

터미널에 위와 같이 입력하면 "my_django_env" 이라는 가상환경이 생성된다. 

 

3-1. 가상환경 다루기

자주 사용하게 될 가상환경 명령어는 👇아래와 같다. 

  • deactivate : 활성화된 파이썬 가상 환경을 비활성화한다
  • workon : 사용가능한 가상 환경 목록을 보여준다
  • workon name_of_environment : 특정 파이썬 가상 환경을 활성화한다
  • rmvirtualenv name_of_environment : 특정 환경을 제거한다.
  • mkvirtualenv name_of_environment : 새로운 가상 환경을 생성한다.

 

수정된 ~/.bashrc 파일이 터미널에 적용되게 하기 위해 우선 터미널을 재시작해준다. 

이러면 자동으로 가상환경이 비활성화된다. 

 

이제 위의 가상환경 명령어 목록으로 가상환경을 생성, 활성화, 비활성화, 삭제할 수 있다. 

~/.zshrc 에서 해당 명령어를 찾을 수 없다는 경고가 가끔 뜨면 그건 터미널 재실행을 안해서 그런거다. ~/.bashrc 파일이나 ~/.zshrc 파일을 수정한 뒤에는 반드시 터미널을 닫고 다시 열어주자. 

 

4. Django 설치

가상환경까지 만들었으면 드디어 가상환경에 Django를 설치해준다. 

workon 가상환경이름 으로 가상환경을 활성화해준 상태에서 아래와 같이 입력한다. 

pip3 install django

pip3를 이용하여 쉽고 간단하게 django를 설치해 줄 수 있다.
